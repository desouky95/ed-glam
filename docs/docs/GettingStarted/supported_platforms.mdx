# Supported platforms

Learn about the platforms, from modern to old, that are supported by ed-glam.

## React

ed-glam supports the most recent versions of React, starting with ^17.0.2 (the one with event delegation at the React root).

## TypeScript

ed-glam requires a minimum version of TypeScript 4.7. This aims to match the policy of DefinitelyTyped, with the support of the versions of TypeScript that are less than two years old.
