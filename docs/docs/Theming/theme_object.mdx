## Usage

To customize theme, pass theme override object to MantineProvider theme prop. Theme override will be deeply merged with default theme.

```tsx
import { ThemeProvider } from '@eduact/student-theme';

function Demo() {
	const [theme, setTheme] = useState<ITheme>({
		colors: {
			primary: '#FFF',
		},

		borderRadii: {
			small: 5,
			medium: 10,
			large: 15,
			xlarge: 20,
		},
		fontFamilies: {
			main: 'Roboto',
		},
	});
	return (
		<ThemeProvider theme={theme}>
			<App />
		</ThemeProvider>
	);
}
```

## Theme Properties

### colors

`theme.colors` ed-glam includes a default color palette out-of-the-box that is a great starting point if you don't have your own specific branding in mind.
You can access the colors through the theme object or or styled components, see the [theme object](/Theming/theme_object) for more information.

### modes

`theme.modes` ed-glam theme provides mode object for different modes themes

```ts
type Mode = 'light' | 'dark' | 'rtl' | 'ltr';
...
modes : {
	'dark': themeObject as ITheme
}
...
```

### buttonColors

`theme.buttonColors` controls RaisedButtons colors based on `theme.colors`.

```ts
buttonColors : {
	'dark' : {
		color: '#FFF';
			background: '#000';
			borderColor: '#000';
	}
}
```

:::note
Any Theme theming object can be used as scale using [styled-system](https://styled-system.com/) variant [see how to use variant](https://styled-system.com/api#variant)

```tsx
import { Color } from '@eduact/student-theme';
import { variant } from 'styled-system';
import styled from 'styled-components';
const Box = styled.div<{ color: Color }>`
	${variant({ scale: 'buttonColors', prop: 'color' })}
`;
```

:::

### textButtonColors

`theme.textButtonColors` used as theming object scale for TextButton component

### backgrounds

`theme.textButtonColors` used as theming object scale for any background usage

### breakpoints

`theme.breakpoints` controls breakpoints used by styled-system which include `'sm' | 'md' | 'lg' |'xl'`

### mediaQueries

`theme.mediaQueries` defines mediaQueries to be used in responsive styles. All mediaQueries works using min-width as ed-glam maintains components as a mobile-first css.

### borderRadii

`theme.borderRadii` used as theming object scale for border-radius for each breakpoint.

### fontSizes

`theme.fontSizes` defines all fontSizes scale

### fontFamilies

`theme.fontFamilies` controls main theme's font-family

### avatarSizes

### avatarShapes

`theme.avatarSizes` & `theme.avatarShapes` used as Avatar component theming object scale

### tableLayout

`theme.tableLayout` controls Tables layout `'fixed' | 'auto'`

### buttonSizes

`theme.buttonSizes` controls any button size accross different breakpoints

### circularProgressColors

### circularProgressSizes

`theme.circularProgressColors` & `theme.circularProgressSizes` used as CircularProgress component theming object scale

### stepperIconSizes

### stepperIconColors

`theme.stepperIconSizes` & `theme.stepperIconColors` used as Stepper component theming object scale

### direction

`theme.direction : DocumentDir` controls theme direction to be used as main html document dir
